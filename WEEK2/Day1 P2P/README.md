# Day 01 – Procure-to-Pay (P2P) Cycle in SAP MM

## Overview

The Procure-to-Pay (P2P) Cycle is a core procurement process in SAP ERP that manages purchasing activities from identifying a requirement to paying the vendor.

Understanding the P2P cycle is essential for SAP professionals because it integrates procurement, inventory management, finance, and vendor management into a single business process.

---

## Learning Objectives

- Understand the end-to-end procurement process.
- Learn the key SAP MM transactions used in procurement.
- Understand how procurement integrates with Finance (FI).
- Build a foundation for SAP MM and SAP Basis administration.
