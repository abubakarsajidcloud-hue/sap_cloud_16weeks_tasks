# SAP Day 2 - ERP Basics

## 📌 Project Overview
This project explains basic SAP ERP concepts including department ownership and system architecture comparison (With and Without ERP).

---

## 🏢 Department Ownership Mapping

| Activity          | Department |
|------------------|------------|
| Buying Steel     | MM         |
| Hiring Employee  | HCM        |
| Paying Vendor    | FI         |
| Selling Car      | SD         |
| Closing Books    | CO         |

👉 This shows how different business activities are handled by SAP modules.

---

## ⚙️ ERP Comparison

### ❌ Without ERP
- Each department works separately
- Separate databases
- No real-time communication
- Data duplication and delays

### ✅ With SAP ERP
- All departments connected to one system
- Central database
- Real-time data sharing
- Better efficiency and decision making
