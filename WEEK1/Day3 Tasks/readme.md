# SAP Day 3 - ERP Module Map

## 📌 Overview
This task shows the basic structure of SAP ERP modules and how they connect to a shared database.

---

## 🧩 SAP Modules Covered

- FI → Financial Accounting  
- CO → Controlling  
- MM → Material Management  
- SD → Sales & Distribution  
- PP → Production Planning  
- QM → Quality Management  
- PM → Plant Maintenance  
- WM → Warehouse Management  
- HCM → Human Capital Management  
- PS → Project System  

---

## 🏗️ SAP ERP Architecture

All modules work inside one SAP ERP system and share a **central database**.

### Key Idea:
- One system
- Many modules
- Single shared database
- Real-time integration
